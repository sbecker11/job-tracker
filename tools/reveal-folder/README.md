# RevealFolder — open a Finder window from a static HTML page

Browsers cannot open Finder from `file://` or ordinary links. This tiny
Mac app registers the custom URL scheme:

```
revealfolder://reveal?path=/absolute/path/to/folder
```

and opens that directory in Finder.

Used by `var/pending-actions.html` company links (generated by
`scripts/render_pending_actions.py`).

## Install (once)

```bash
cd job-tracker/tools/reveal-folder
./install.sh
```

Installs `~/Applications/RevealFolder.app` and registers the scheme with
Launch Services. No Dock icon (LSUIElement).

## Smoke test

```bash
open 'revealfolder://reveal?path='"$HOME"'/Desktop'
```

Or click any company name in `pending-actions.html` after re-rendering:

```bash
python scripts/render_pending_actions.py --no-rescore
```

The first browser click may show “Open RevealFolder?” — allow it.
